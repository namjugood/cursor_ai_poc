# OK Capital Cursor AI PoC
Cursor AI를 활용한 화면 및 프론트엔드 코드 생성 PoC

## PoC 목표
* Cursor AI를 이용한 퍼블리싱 및 프론트엔드 구현 자동화 가능 여부 확인
* Cursor AI를 이용하여 업무 활용도 확인 및 사용 방안 도출

## PoC 세부내용

|프레임워크|v4.18.2|Express.js|
